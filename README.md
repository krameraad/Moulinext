# Moulinext
### Work in progress!
A testing suite for Libft of the 42 curriculum.
*Moulinext* runs a few basic tests for the project itself,
then runs tests for all functions in Libft.

After all tests are run, it reports a list of errors found
with a short description explaining it's requirements for success.
